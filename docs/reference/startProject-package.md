# startProject: A package for creating a project directory

The start.project package can be used to create a project directory and
generate memo and code templates.

## startProject Package Functions

Functions to create a project directory with sub-folders and templates:

[`startProject`](https://github.com/CU-Anschutz-DOS-CORP/startProject/reference/startProject.md):
A function to create project directory.

[`runApp_startProject`](https://github.com/CU-Anschutz-DOS-CORP/startProject/reference/runApp_startProject.md):
A function to launch
[`startProject`](https://github.com/CU-Anschutz-DOS-CORP/startProject/reference/startProject.md)
as a local Shiny app.

Functions to create additional templates:

[`makeMemoTemplate`](https://github.com/CU-Anschutz-DOS-CORP/startProject/reference/makeMemoTemplate.md):
A function to generate a .docx memo template.

[`makeRmdTemplate`](https://github.com/CU-Anschutz-DOS-CORP/startProject/reference/makeRmdTemplate.md):
A function to generate a .Rmd template.

[`makeRTemplate`](https://github.com/CU-Anschutz-DOS-CORP/startProject/reference/makeRTemplate.md):
A function to generate a .R template.

[`makeSasTemplate`](https://github.com/CU-Anschutz-DOS-CORP/startProject/reference/makeSasTemplate.md):
A function to generate a .sas template.
